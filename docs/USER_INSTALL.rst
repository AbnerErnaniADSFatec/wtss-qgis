User Installation
=================


download the zip file for `wtss_plugin`

Or if you download to install by `zip` file, use:

.. code-block:: text

    $ pip3 install git+https://github.com/brazil-data-cube/wtss-qgis.git@v0.3.0

.. code-block:: text

    \> pip3 install git+https://github.com/brazil-data-cube/wtss-qgis.git@v0.3.0
